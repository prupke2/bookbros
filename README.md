# BookBros

Book Bros provides an easy way to keep track of and rate books in your book club.

## Book Search

Search for books using Google Books API, with book covers, summaries, and ratings. Selecting a book adds it to your club's list.

![Book Search](main/public/readme_files/book_search.png)
